

# Project
## Starter
- 启动器
    - 最简化的 Spring.boot 启动


## Seed
- 种子项目
	- 平时使用的基础项目结构    

## Module
- Security
	- memory
		- 内存形式配置用户信息
		- 配置接口的访问角色对应权限
	- jdbc
		- 用户数据来源于数据
		- 支持 H2 用户数据初始化
	- redis
		- 支持 redis 存储 spring session 信息
	- oauth
		- 支持 oauth 的客户端模式和密码模式
